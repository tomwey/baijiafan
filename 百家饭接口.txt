Host http://120.132.57.133/api/v1

备注：每个接口调用会返回message，建议可以使用message的内容进行有好的接口返回提示

1.获取验证码
POST /auth_codes/fetch
参数  mobile 手机号，必须

2.用户登录
POST /account/login
参数  mobile 手机号，必须
     code 短信验证码，必须
     
3.修改用户个人资料
POST /user/update_profile
参数  token 登录成功后，会返回Token, 必须
     nickname 用户名，可选参数
     avatar 图片数据，可选参数
